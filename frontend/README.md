# Frontend

React + TypeScript + Vite UI for SimplifAI.

## Scripts

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run lint
```

Set `VITE_API_BASE_URL` in `.env` (defaults to `http://localhost:8000/api/v1`).

See the root [README.md](../README.md) for full project setup.
